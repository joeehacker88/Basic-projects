Network Troubleshooting

No Internet Connection

  1. Check if the Ethernet cable or Wi-Fi is connected properly.
  2. Restart modem and router (unplug 30s).
  3. Run commands: 
    Windows: ipconfig /release && ipconfig /renew && ipconfig /flushdns
Linux: sudo systemctl restart NetworkManager
  4. Try accessing a website by IP (http://8.8.8.8).

Slow Internet

  • Disconnect unused devices.
  • Run a speed test (speedtest.net).
  • Switch to Ethernet if possible.

Software Installation

Cannot Install Software

  1. Verify you have admin rights.
  2. Check disk space.
  3. Ensure antivirus/firewall isn’t blocking installer.

Application Crashing

  • Update to the latest version.
  • Reinstall application.
  • Check Event Viewer (Windows) or journalctl (Linux).

Printer Issues

Print Jobs Stuck

Windows:
net stop spooler
del /Q /F %systemroot%\System32\spool\PRINTERS\*
net start spooler

Linux:
sudo systemctl restart cups

Printer Not Detected

  • Ensure printer is on and connected.
  • Reinstall drivers.
  • Verify printer IP address.

System Performance Tuning

Computer Running Slow

  • Check Task Manager (Windows) or htop (Linux).
  • Uninstall unused programs.
  • Disable startup applications.
  • Run disk cleanup or clear caches.

Frequent Freezes

  • Update drivers and OS patches.
  • Run hardware diagnostics.

Useful Tools

  • Windows: sfc /scannow, chkdsk /f
  • Linux: dmesg, journalctl -xe
